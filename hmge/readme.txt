Harbour MiniGUI comprises these main components:

	- Harbour MiniGUI Extended Library
	- Harbour MiniGUI IDE
	- Harbour Compiler
	- Borland C++ Compiler and command-line tools at
	http://forms.embarcadero.com/BCC32CompilerDownload

	or go directly to the download form at
        https://www.embarcadero.com/free-tools/ccompiler/free-download

	Take a look for the useful information also at
	http://rudeserver.com/how-to/install-borland/

There are also the private MiniGUI builds for the following C-compilers:

	- MinGW GNU C 9.1.1 (32-bit);
	- MinGW GNU C 8.2.0 (64-bit);

	- MS Visual C++ 2019 (32-bit);
	- MS Visual C++ 2019 (64-bit);

	- Borland/Embarcadero C++ 7.4 (32-bit);

	- Open Watcom C/C++ 2.0 (32-bit);

	- Pelles ISO C 9.0 (32-bit);
	- Pelles ISO C 9.0 (64-bit).

When used together, these tools produce a free xBase Win32 GUI development
system, continuing the xBase tradition but with an extremely easy to use GUI.
